# **App Name**: Seyara Yabi

## Core Features:

- Car Price Estimation: Estimate car market value based on user inputs like brand, model, year, and condition.
- Price Estimation Calculations: Calculate minimum, average, and maximum estimated prices based on available market data.
- Car Listing Creation: Allow users to create car listings with detailed specifications and images.
- Car Listing Display: Display car listings with filtering by brand, model, year, location, and other specifications.
- AI Negotiation Tool (Simulation): Provide a simulated AI negotiation tool to assist users in buying or selling cars.
- Market Insight Display: Display market insights such as active listings, demand areas, average selling time, and supply/demand ratios. Uses the tool to surface insights based on the filters applied.
- User Messaging: Enable private in-app messaging for communication between buyers and sellers.

## Style Guidelines:

- Primary color: Dark teal (#0B2E3D) to establish a professional feel.
- Accent color: Seafoam green (#1EC9A8) for interactive elements and highlights.
- Background color: Off-white (#F5F9FA) to provide a clean backdrop for content.
- Font: 'PT Sans', a humanist sans-serif, for both headlines and body text.
- Use crisp, modern icons to represent car types, filters, and features.
- Maintain a clean, card-based layout for car listings and market data display.
- Use subtle transitions for loading data and displaying search results.
